# Jeju-Olle 
### 제주도의 전기차 충전소 안내, 맛집안내, 숙소 안내, 리뷰 및 별점, 제주도 굿즈 판매 및 결제 서비스 및 로그 데이터 분석을 목표로 한다

- 숙박시설
- 맛집
- 상품구매
- 실시간 검색 순위
